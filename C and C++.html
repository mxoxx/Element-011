Coding for Everyone: C and C++ Specialization





What does this course offer?:

This course is for all programming enthusiasts, as well as beginners, computer and other scientists, and artificial intelligence enthusiasts seeking to develop their programming skills in the foundational languages of C and C++. Through the four courses — two in C, and two in C++ — you will cover the basics of programming in C and move on to the more advanced C++ semantics and syntax. This will prepare you to apply these skills to a number of higher-level problems using AI algorithms.

You will learn to:

  • Write and debug code in C and C++ programming languages.
  • Understand algorithms, and how to properly express them.
